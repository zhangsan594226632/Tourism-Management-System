# Tourism-Management-System
【万字文档+源码】基于springboot+vue旅游管理系统-可用于毕设-课程设计-练手学习-学习资料分享-本文介绍了一个基于SpringBoot+Vue前后端分离架构开发的旅游管理系统。系统分为游客前台和管理员后台两大模块，旨在解决传统文旅行业信息滞后、预订繁琐等问题。游客可浏览景点、购买门票、发布攻略、收藏景点；管理员可维护景点分类、资讯、用户数据等。技术栈采用SpringBoot（MyBatis-Plus、JWT鉴权）和Vue3（Element Plus、Axios），数据库使用MySQL 8.0。系统通过角色划分实现业务流程，包含登录注册、景点展示、购票、攻略管理等核心功能模块，并详细设计了用户表、景点表
